# Frontend (Vite + React)

## Stack
- Vite + React + TypeScript
- Tailwind CSS
- React Router
- React Query
- Firebase client SDK

## Run locally
1. Install dependencies:
   - `npm install`
2. Copy `.env.example` to `.env` and set values.
3. Start dev server:
   - `npm run dev`

Backend default base URL: `http://localhost:8000/api/v1`
